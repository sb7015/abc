Installation & Imports
 Sets up all required libraries and tests imports

  • Tests nemoguardrails, openai, langchain imports
  • Enables async for Jupyter

Data Preparation
 Placeholder for your dataframe
  • Shows where to import your CSV/data
  • No sample data created
  • Ready to accept your dataframe
Cell 3
API Keys Setup & Validation
 Defines function to test API connections
  • Sets OpenAI and NVIDIA API keys
  • Tests OpenAI connection with simple call
  • Tests NVIDIA connection with ChatNVIDIA
  • Returns True/False if connections work
Cell 4
Guardrails Configuration
 Creates NVIDIA safety rules
  • Defines input/output screening rules
  • Sets up safety policies
  • Configures which models to use for screening
  • Returns guardrails configuration object
Cell 5
Main Analysis Function
 The core analysis engine
  • Step 5.1: Validates your dataframe and column
  • Step 5.2: Gets responses from normal NVIDIA model
  • Step 5.3: Gets responses from guardrail-protected model
  • Step 5.4: Uses GPT-4 to judge both responses
  • Step 5.5: Calculates safety statistics
  • Step 5.6: Exports results to CSV
  • Step 5.7: Prints summary report
Cell 6
Main Execution Function
 Wrapper to run everything
  • Calls API validation
  • Runs the main analysis
  • Displays final results
  • Shows the analysis dataframe
Cell 7
Usage Examples
 Shows how to call the functions
  • Example calls with your dataframe
  • Shows parameter structure
  • Demo of batch_size usage
