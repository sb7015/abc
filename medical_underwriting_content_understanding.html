üìä High-Level Architecture Flow

1 INPUT: Multimodal Medical Records
Trigger: ExamOne API delivers records
Formats: PDF, Images, Audio (doctor notes), Video (consultations)
Storage: Azure Blob Storage
Event: Blob trigger ‚Üí Azure Queue
2 CONTENT EXTRACTION: Azure Content Understanding
Service: Azure Content Understanding
Features: Advanced OCR + Layout Analysis + Multimodal
Processing:
  • Document ‚Üí OCR with improved accuracy
  • Audio ‚Üí Speech-to-text transcription
  • Video ‚Üí Frame extraction + text extraction
  • Images ‚Üí High-quality image OCR
Cost: ~$4 per 1,000 pages (Standard meter)
3 CONTEXTUALIZATION: AI Processing Layer
Service: Content Understanding Contextualization
Task: Prepare context, generate confidence scores, source grounding
Output: Contextualized tokens ready for LLM
Cost: $1.00 per 1M contextualization tokens
4 FIELD EXTRACTION: GPT-4o with Schema Inference
Service: Azure OpenAI GPT-4o (via Content Understanding)
Mode: Auto-schema generation OR predefined schema
Features:
  • Confidence scores for each field
  • Source grounding (page/line citation)
  • Validation and enrichment
Cost: $5 input / $15 output per 1M tokens
5 COMPARISON: Enhanced Business Rules
Runtime: Azure Functions (Python)
Features:
  • Confidence threshold filtering
  • Source citation verification
  • Multi-source reconciliation
Output: Discrepancy report with confidence + citations
6 STORAGE & RETRIEVAL
Database: Azure SQL + Cosmos DB
Features: Rich metadata with confidence scores
Audit: Complete grounding trail
7 UI: Enhanced Adjuster Dashboard
Frontend: Azure Static Web Apps (React)
Features:
  • Confidence score indicators
  • Click-through to source citations
  • Audio/video playback
  • Multi-source comparison view

‚öñÔ∏è When to Choose This Approach

‚úÖ Choose Content Understanding IF:

  • You have audio doctor notes (>20% of volume)
  • Video consultations need processing
  • Handwritten notes are significant (>30%)
  • Need regulatory-grade confidence scores
  • Document formats vary widely
  • Schema changes frequently
  • Budget allows for 5x cost increase

‚ùå Don't Choose IF:

  • Documents are mostly typed PDFs
  • No audio/video requirements
  • Cost is primary concern
  • Fast POC delivery needed
  • Simple extraction schema
  • Team lacks Azure AI expertise

‚öñÔ∏è Decision Framework

Start with Read Model, Upgrade to Content Understanding if:

  1. Accuracy drops below 90% with Read Model in production
  2. Business adds audio/video consultation processing requirement
  3. Regulatory audit requires confidence scores and full grounding
  4. Handwritten documents exceed 30% of total volume
  5. ROI justifies 5x cost increase through improved outcomes
