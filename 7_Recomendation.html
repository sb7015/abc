游눯 Cost Efficiency Calculation

Value Score Formula:

Value Score = (Accuracy 칑 Uptime 칑 Detection Coverage) / (Latency 칑 False Positive Rate 칑 Cost per 1M requests)

Example Calculation:

Guardrail Solution A:

  • Accuracy: 98%
  • Uptime: 99.9%
  • Detection Coverage: 10 threat types
  • Latency: 45ms
  • False Positive Rate: 2%
  • Cost: $50 per 1M requests
Value Score = (98 칑 99.9 칑 10) / (45 칑 2 칑 50) = 9,790.2 / 4,500 = 2.18

Interpretation:

  • 游릭 High Value: Score > 2.0
  • 游리 Medium Value: Score 1.0-2.0
  • 游댮 Low Value: Score < 1.0
