üí° Use Case: Automated Medical Record Summarization for Life Insurance Underwriting
üéØ Goal: Extract medical data from PDFs, compare with application, flag discrepancies

üìä High-Level Architecture Flow

1 INPUT: Medical Records Arrive
Trigger: ExamOne API delivers medical records
Storage: Azure Blob Storage (PDF files)
Event: Blob trigger sends message to Azure Queue
2 OCR: Azure Document Intelligence (Read Model)
Service: Azure AI Document Intelligence
Model: prebuilt-read
Output: Full text extraction from all pages
Cost: $1.50 per 1,000 pages
3 EXTRACTION: GPT-4o-mini Structured Entity Extraction
Service: Azure OpenAI GPT-4o-mini
Task: Extract diagnoses, medications, dates, ICD-10 codes
Output: Structured JSON with medical entities
Cost: $0.15 input / $0.60 output per 1M tokens
4 COMPARISON: Python Business Rules Engine
Runtime: Azure Functions (Python)
Task: Compare application vs medical data
Logic: Predefined business rules (no AI needed)
Output: Discrepancy report with severity flags
5 STORAGE: Save Results
Database: Azure SQL (structured data)
Storage: Cosmos DB (discrepancy reports)
Purpose: Audit trail, historical analysis
6 UI: Adjuster Dashboard
Frontend: Azure Static Web Apps (React)
Display: Flagged discrepancies with citations
Actions: Approve, Deny, Request Clarification
